# Football Ticket Booking System
Football Ticket Booking System manages users, matches, and match ticket bookings. 

The database has 3 tables (Users, Matches, Bookings)
The Users table contains all the information about the users, the Matches table contains the match information, and the Bookings table contains all the booking information.
These tables has relationship with each other through foreign keys.
